# Registration Form with Real-Time Validation

A simple and responsive Registration Form built using **HTML, CSS, and JavaScript**. This project performs real-time form validation to ensure users enter valid information before submitting the form.

## Features

* Real-time input validation using JavaScript
* Name field validation (cannot be empty)
* Email format validation
* Password validation (minimum 6 characters)
* Error messages displayed for invalid inputs
* Register button remains disabled until all fields are valid
* Success message displayed after successful registration
* Clean and responsive user interface using HTML and CSS

## Technologies Used

* HTML5
* CSS3
* JavaScript
